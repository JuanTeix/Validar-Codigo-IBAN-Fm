
# Validación de Códigos IBAN con FileMaker

## Descripción
Este repositorio contiene un tutorial completo y código de ejemplo para validar códigos IBAN (Número de Cuenta Bancaria Internacional). El propósito de este tutorial es proporcionar a los desarrolladores, analistas financieros y cualquier persona interesada en procesos bancarios internacionales, las herramientas y conocimientos necesarios para validar códigos IBAN, asegurando que las transacciones financieras se realicen sin errores y de manera eficiente.


## Contenidos
1. **Introducción al IBAN**
   -  Comprender la estructura y el propósito de los números IBAN.
2. **Desglose del IBAN**
   - Cómo se estructuran los diferentes componentes de un IBAN.
3. **Método de Validación**
   - Explicación detallada de cómo validar un IBAN utilizando el algoritmo mod97.
4. **Código de Ejemplo**
   - Scripts listos para usar como función personalizada.

## Cómo Usar Este Tutorial
- Para aprovechar al máximo este tutorial, te recomendamos seguir los pasos en el orden presentado. Comienza con la introducción para obtener una base sólida sobre lo que es un IBAN y por qué es importante su validación. Después, profundiza en la sección técnica para entender cómo puedes implementar la validación por ti mismo.
